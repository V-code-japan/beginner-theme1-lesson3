# LESSON3 倉庫を作ってみよう
```block
player.onChat("run", function () {
    blocks.fill(
    PLANKS_OAK,
    world(0, 0, 0),
    world(0, 0, 0),
    FillOperation.Replace
    )
    blocks.place(GRASS, pos(0, 0, 0))
})
```

## 目標
<p>まずはこれから作るものの形を確認しよう！</p>
<p>・チャットで「run」と入力したら、倉庫を作るようにプログラミングをする</p>
<p>・石レンガをワールド(24, 74, 134)からワールド(30, 79, 140)に中空（中が空どう）で置く</p>
<p>・ガラスをワールド(26, 79, 136)からワールド(28, 79, 138)に置く</p>
<p>・トウヒのドアをワールド(24, 75, 137)に置く</p>

## 1.コードブロックを設置してみましょう

`||blocks.ブロック||`ボタンをクリックしてその中の`||blocks.ブロックを並べる～||`と書いてあるブロックを`||player.チャットコマンド～||`の中にドラックしてみましょう</br>
`||blocks.土ブロック||`をクリックして`||blocks.石レンガ||`に変えましょう

## 2.座標を変更してみましょう

`||positions.ポジション||`ボタンをクリックしてその中の`||positions.ワールド0 0 0||`を`||blocks.範囲の始まり~0~0~0||`の上までドラッグします</br>
`||positions.範囲の始まり ワールド0 0 0||`を`||positions.ワールド24 74 134||`に変えましょう</br></br>
`||positions.ポジション||`ボタンをクリックしてその中の`||positions.ワールド0 0 0||`を`||blocks.範囲の終わり~0~0~0||`の上までドラッグします</br>
`||positions.範囲の終わり ワールド0 0 0||`を`||positions.ワールド30 79 140||`に変えましょう</br>

## 3.設定を変更してみましょう

`||blocks.モード 置き換え(replace)||`をクリックして`||blocks.モード 中空||`に変えましょう</br>

## 4.まどを作りましょう

`||blocks.ブロック||`ボタンをクリックしてその中の`||blocks.ブロックを~0~0~0に並べる||`と書いてあるブロックを`||player.チャットコマンド～||`の中にドラックしましょう</br>
`||blocks.土ブロック||`をクリックして`||blocks.ガラス||`に変えましょう</br>

## 5.座標を変えてみましょう

`||positions.ポジション||`ボタンをクリックしてその中の`||positions.ワールド0 0 0||`をドラックして`||blocks.範囲の始まり~0~0~0||`の上までドラッグします</br>
`||positions.範囲の始まり ワールド0 0 0||`を`||positions.ワールド26 79 136||`に変えましょう</br></br>
`||positions.ポジション||`ボタンをクリックしてその中の`||positions.ワールド0 0 0||`をドラックして`||blocks.範囲の終わり~0~0~0||`の上までドラッグします</br>
`||positions.範囲の終わり ワールド0 0 0||`を`||positions.ワールド28 79 138||`に変えましょう</br>

## 6.ドアをおいてみましょう

`||blocks.ブロック||`ボタンをクリックしてその中の`||blocks.ブロックを~0~0~0に置く||`と書いてあるブロックを`||player.チャットコマンド～||`の中にドラックしてみましょう</br>
`||blocks.土ブロック||`をクリックして`||blocks.トウヒのドア||`に変えましょう

## 7.座標を変えてみましょう
